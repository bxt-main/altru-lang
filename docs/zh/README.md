# altru 编程语言 - 中文文档

## 项目背景

altru 是一个实验性编程语言项目，由非程序员通过AI辅助开发。本项目的主要目的是邀请专业编程语言开发者进行技术评估，并就各种开放性问题进行讨论。

## 为什么创建 altru？

- **探索AI在编程语言开发中的应用**：测试AI如何协助非程序员设计和实现编程语言
- **学习编程语言理论**：通过实践理解编译器、解释器和语言设计的基本原理
- **社区协作**：希望借助开源社区的力量，获得专业开发者的指导和建议

## 项目状态

这是一个非常早期的实验性项目。代码质量、架构设计和功能实现都可能存在严重不足。我们诚恳地寻求专业开发者的反馈和帮助。

## 如何参与

### 技术评估
- 审查现有代码和架构
- 提出改进建议
- 指出潜在的问题和风险

### 设计讨论
- 讨论语言特性设计
- 建议语法和语义改进
- 分享编程语言设计的最佳实践

### 代码贡献
- 修复bug
- 实现新功能
- 改进性能和稳定性

## 重要说明

由于作者并非专业程序员，项目中可能存在：
- 不符合最佳实践的代码
- 效率低下的算法
- 架构设计缺陷
- 安全漏洞

我们完全理解这些问题的存在，并希望通过社区的帮助逐步改进。
