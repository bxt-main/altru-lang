## 🌐 多语言支持

本项目采用双语文档策略：
- **所有新文档必须同时提供中文和英文版本**
- **中文为主，英文为辅**
- **保持内容同步更新**

文档结构：
```
docs/
├── README.md              # 本文档导航页
├── zh/                    # 中文文档目录
│   ├── overview.md        # 概述与设计理念
│   ├── specification-0.2.1.md  # 语法规范
│   ├── design-principles.md    # 设计原则
│   ├── language-comparison.md  # 语言对比
│   ├── ai-integration.md       # AI集成特性
│   └── development-guide.md    # 开发指南
└── en/                    # 英文文档目录
    ├── overview.md        # Overview & Design Philosophy
    ├── specification-0.2.1.md  # Syntax Specification
    ├── design-principles.md    # Design Principles
    ├── language-comparison.md  # Language Comparison
    ├── ai-integration.md       # AI Integration Features
    └── development-guide.md    # Development Guide
```

## 📝 贡献指南

当添加新文档时，请遵循以下步骤：

1. **在 `zh/` 目录下创建中文版本**
2. **在 `en/` 目录下创建对应的英文版本**
3. **更新本导航页的链接列表**
4. **确保两个版本内容保持同步**

> **注意**: 中文文档是主要版本，英文文档应准确反映中文内容。
