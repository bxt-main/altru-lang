# altru Language Specification

This directory contains the official language specifications for altru programming language.

## Versions

- **v0.2.1** - Current specification version
- **draft/** - Work-in-progress specifications

## Structure

Each specification version includes:
- `design-principles.md` - Core design philosophy
- `language-features.md` - Detailed language features
- `syntax-reference.md` - Complete syntax reference
- `examples/` - Code examples demonstrating features
